# sfTestToolforCommunity

Seleniumでテストコードを書くためのベース。  
以下ののWebdriverに対応。  

-  1)chrome  
-  2)firefox  

/lib/Utilityにいくつかメソッドがあります。  
スクロール、スクリーンショット、クリックなど。  

随時追加予定。  
適当なフォルダに移動。  

1) `clone {repositry}`  
2) `npm install`  
3) `node ./src/sample.js chrome`  
4) `node ./src/sample.js firefox`
